# cypress-automacao

# Instalação

### Utilize os comandos abaixo para utilizar o projeto corretamente

</br>
Faça o clone do projeto via HTTPS:

https://github.com/william-pereira/teste-pratico.git

```
npm install
```

### Após todas as instalações rode o comando

```
npm run cy_open
```
